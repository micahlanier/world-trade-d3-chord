Visualizing World Trade

The following chord diagram was produced as part of a homework assignment for CS 171 at Harvard. It depicts countries’ top ten export destinations from 1995 to 2012, and allows interaction with filters and hovering effects. Colors reflect each country’s associated continent. A large display is recommended.

A chord diagram is likely not the best option for displaying this information and the data source is imperfect. But it proved a great chance to learn more about Javascript data manipulation and D3 transitions. Source is available on GitHub; D3 developers may find the approach to data manipulation and animated transitions useful.
